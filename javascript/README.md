# learn-nodejs

## 想玩凯撒的游戏吗？

将源码下载下来后，打开 `test.html` 就能玩到精彩刺激的凯撒游戏咯。  

由于安全限制，需要将 `test.html` 用 `http` 之类的协议打开，所以得先部署到本地的服务器上，推荐使用 `VS Code` 的 `Live Server` 插件。  

玩得开心：）
